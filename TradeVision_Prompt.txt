## WHAT THIS APP DOES

TradeVision AI is a quantitative trading dashboard that:
- Runs backtests on historical stock/ETF data using 3 different strategies
- Displays equity curves, drawdown charts, and performance metrics (Calmar, Sharpe, Win Rate)
- Shows daily BUY / SELL / HOLD trading signals with explanations
- Lets users configure strategy parameters (lookback, position size, risk controls)
- Built for the Builderr.ai Trading Agent Challenge (scored by Calmar Ratio)

## 5 API ENDPOINTS (Backend)

```
GET  /health              → { "status": "ok" }

POST /backtest            → Run strategy simulation
  Body: { strategy, tickers[], start_date, end_date, initial_cash }
  Returns: { equity_curve[], trades[], dates[], drawdown_series[], metrics{} }

GET  /prices/{ticker}     → Fetch OHLCV price bars
  Query: ?period=1y
  Returns: [{ ts, open, high, low, close, volume }]

GET  /strategies          → List available strategies + params

POST /simulate/live       → Get today's signal
  Returns: { signal, ticker, reason, suggested_orders[] }
```

## 3 TRADING STRATEGIES

### Dual Momentum (Default)
- Rank ETFs by 3-month return, buy top 3 with positive momentum
- Risk-off: If SPY < 200-day SMA → sell everything, go to cash
- Universe: SPY, QQQ, IWM, XLK, XLF, XLE, XLV, GLD
- Position sizing: Equal weight, max 30% per position

### SMA Crossover
- Buy on Golden Cross (SMA-50 > SMA-200)
- Sell on Death Cross (SMA-50 < SMA-200)
- Simple trend-following, few trades

### Volatility Target
- Size positions inversely to their 20-day volatility
- Target 10% annualized portfolio volatility
- Cap leverage at 1.5x

## KEY METRICS (displayed in MetricCards)

| Metric | Formula | Good Value |
|--------|---------|------------|
| Calmar Ratio | Ann. Return / Max Drawdown | > 2.0 |
| Sharpe Ratio | (Return - Rf) / StdDev | > 1.0 |
| Sortino Ratio | (Return - Rf) / Downside StdDev | > 2.0 |
| Max Drawdown | (Peak - Trough) / Peak | < 20% |
| Win Rate | Winning Trades / Total Trades | > 50% |
| Profit Factor | Gross Profit / Gross Loss | > 1.5 |
